Objetivo de la práctica
Aplicar los fundamentos de la Programación Orientada a Objetos en PHP mediante la creación de una clase con atributos privados, utilizando encapsulamiento y métodos de acceso (getters y setters).

Descripción de la clase creada
Se creó una clase llamada Usuario que representa a un usuario del sistema.
La clase contiene dos atributos privados: nombre y correo.

Para trabajar con estos atributos se implementaron los siguientes métodos:
Un constructor para inicializar los valores del usuario al crear el objeto.
Métodos getNombre() y getCorreo() para obtener los valores de los atributos.
Métodos setNombre() y setCorreo() para modificar los valores de los atributos.
El uso de atributos privados permite aplicar el principio de encapsulamiento, evitando que los datos sean modificados directamente desde fuera de la clase.

Instrucciones de ejecución
Instalar y ejecutar el servidor local mediante XAMPP.
Colocar la carpeta del proyecto dentro del directorio htdocs.
Iniciar el servidor Apache desde el panel de control de XAMPP.
Abrir un navegador web.

Acceder a la siguiente dirección:
http://localhost/desarrollo-web-avanzado-fimaz-uas/parcial-1-poo/practica-1/

El archivo index.php ejecutará la clase Usuario y mostrará en pantalla el nombre y el correo del usuario utilizando los métodos getters.
